# newsletter-automation

HRD 관련 뉴스와 아티클을 수집하고, AI를 활용해 뉴스레터형 콘텐츠로 재작성한 뒤 HTML 웹페이지로 생성하는 Streamlit 기반 자동화 프로젝트입니다.

## 주요 기능

- HRD 뉴스/아티클 데이터 관리
- 뉴스레터용 기사 선별
- AI 기반 뉴스레터 요약 콘텐츠 생성
- HTML 웹페이지 렌더링
- Streamlit 관리자 화면에서 미리보기 및 HTML 확인
